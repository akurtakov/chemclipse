Prefixes

Mass Spectrum Search

name: Searches for the compound name.
ref:
id:		 Searches the reference or internal identifier for the record as in the database ID or the accession number.
formula: Searches the molecular formula field.
smiles: Searches the SMILES structure code.
inchi: Searches the InChI structure code.
inchikey: Searches the InChIKey (hashed InChI) field.
cas: Searches the CAS registry number for the compound.
comment:
comments:		 Searches notes attached to the record by the curator.
synonym:
synonyms:		 Searches synonyms and alternate names for the compound.
database: Searches the source database name.
odor: Searches odor descriptor fields.
matrix: Searches the food matrix context in terms of flavor.
solvent: Searches solvent information attached to sensory/flavor records.
all: Searches across all searchable fields which is also the default when no key is used.
