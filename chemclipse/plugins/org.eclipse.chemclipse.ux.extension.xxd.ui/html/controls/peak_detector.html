Peak Detector

Use a hotkey (see below) or the toolbar button to set a mode. Confirm the peak with the add button or its hotkey.
Key Description
D B: Baseline
Use CTRL and drag with the mouse.
B BB: Baseline to Baseline
Use left click to define start and stop.
V VV: Valley to Valley
Use left click to define start and stop.
N BV: Baseline to Valley
Use left click to define start and stop.
C VB: Valley to Baseline
Use left click to define start and stop.
A +: Add the peak.
