Data Explorer Icons

The following icons can appear:

Chromatography

Icon Description
CSD: current selective detector such as GC-FID, -ECD
MSD: mass selective detector such as GC-MS/LC-MS
TSD: time selective detector such as GCxGC
VSD: vibrational spectroscopy selective detector such as GC-NIR
WSD: wavelength selective detector such as HPLC-DAD or -UV-Vis

Spectroscopy

Icon Description
NMR: nuclear magnetic resonance spectroscopy
FID: free induction decay in NMR
FSD: fluorescence spectroscopic detector as in fluorimetry
VSD: vibrational spectroscopic detector such as FT-IR

Molecular Biology

Icon Description
PCR: polymerase chain reaction (real-time)
