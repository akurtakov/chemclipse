S Plot

Left Mouse Button down Start Zoom selection
Left Mouse Button up Stop Selection Zoom
Ctrl(Win/Linux)/CMD(Mac) + Left Mouse Button down Start selection of features to be highlighted in Features list
Ctrl(Win/Linux)/CMD(Mac) + Left Mouse Button up Stop selection of features to be highlighted in Features list
Double Click Select the nearest feature
Ctrl(Win/Linux)/CMD(Mac) + Double Click Add or remove the nearest feature from the current selection
